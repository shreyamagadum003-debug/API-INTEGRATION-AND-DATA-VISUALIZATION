## Weather API & Data Visualization App

This Python application fetches real-time weather data for a specified city using the OpenWeatherMap API and visualizes key weather parameters in a column/bar graph. The visualization provides a quick overview of the city’s temperature, feels-like temperature, and humidity, making it easy to understand current weather conditions at a glance.

## Features

Fetches live weather data for any city using OpenWeatherMap API.

Extracts important parameters: Temperature (°C), Feels Like (°C), and Humidity (%).

Displays data in a clear and intuitive column graph using Matplotlib and Seaborn.

Graph includes proper X and Y axis labels, title, and gridlines for readability.

Easy to extend: additional weather parameters or multiple cities can be added for comparison.

## Installation

Clone the repository:

git clone <repository_link>


## Install required libraries:

pip install requests pandas matplotlib seaborn


Get a free API key from OpenWeatherMap
.

## Usage

Open weather_visualization.py in Python or VS Code.

Replace "YOUR_API_KEY_HERE" with your API key.

Set the desired city name in the CITY variable.

## Run the script:

python weather_visualization.py

A column graph will display the weather parameters for the selected city.
